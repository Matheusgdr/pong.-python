# Pong em Python com Pygame

Meu primeiro projeto de jogo utilizando Python e Pygame.

O objetivo deste projeto foi praticar conceitos fundamentais de programação enquanto construía um jogo clássico do zero, sem utilizar engines prontas.

## O que foi desenvolvido

* Sistema de classes utilizando Programação Orientada a Objetos (POO)
* Barra controlada pelo jogador
* Barra adversária com movimentação automática
* Bola com movimentação em dois eixos
* Colisões com paredes e barras
* Sistema de pontuação
* Reinício automático da bola após cada ponto
* Controle de FPS utilizando `pygame.time.Clock()`

## Conceitos praticados

* Classes e objetos
* Métodos e atributos
* Uso de `self`
* Estruturas condicionais
* Operadores lógicos (`and`, `or`)
* Geometria e coordenadas em jogos 2D
* Detecção de colisões
* Organização de código
* Manipulação de eventos do teclado
* Renderização de texto e placar

## Tecnologias utilizadas

* Python 3
* Pygame
